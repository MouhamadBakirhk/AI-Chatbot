# Projectauce - AI Chatbot

AI Chatbot project built using PHP (Laravel) and Python (Flask).

## Overview
This project is an AI chatbot that communicates between a Laravel backend and a Python Flask API to generate responses.

## Architecture
- Laravel handles the web interface and requests
- Flask handles AI processing and chatbot logic
- Communication between both via API

## Technologies Used
- PHP (Laravel)
- Python (Flask)
- REST API
- HTML / CSS / JavaScript

## Features
- Chat interface
- AI-powered responses
- Backend API integration (Laravel ↔ Flask)
- Modular architecture

## Purpose
This project was built to practice full-stack development with AI integration and API communication between different frameworks.
